# Movie Recommendation System

## Project Overview
This project implements a Movie Recommendation System that suggests movies to users based on their previous ratings and preferences. The system uses collaborative filtering techniques to analyze user behavior and generate personalized recommendations.

The project was developed using Python in Google Colab.

---

## Objective
The main objective of this project is to build a recommendation system that:
- Analyzes user movie ratings
- Identifies similar users
- Recommends movies based on user preferences

---

## Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Scikit-learn

---

## Dataset
The system uses the MovieLens dataset containing:
- User ID
- Movie ID
- Rating
- Timestamp

---

## Methodology
1. Load and preprocess dataset
2. Create user–movie matrix
3. Calculate similarity using cosine similarity
4. Generate personalized movie recommendations

---

## How to Run
1. Open the notebook in Google Colab.
2. Run all cells sequentially.
3. Enter a User ID to generate recommendations.

---

## Output
The system recommends movies based on similarities between users’ rating patterns.
